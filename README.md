# 🚀 بوت استضافة تلجرام الاحترافي (جاهز لـ Railway)

هذا المشروع جاهز للرفع والتشغيل الفوري على منصة Railway أو أي VPS.

## 📁 محتويات المشروع:
- `main.py`: الكود الأساسي للبوت مع نظام الاستضافة الحقيقي.
- `database.py`: نظام إدارة قاعدة البيانات (يتم إنشاؤه تلقائياً).
- `requirements.txt`: المكتبات اللازمة للتشغيل.
- `Procfile`: ملف الإعداد لمنصة Railway.

## 🚀 كيفية الرفع على Railway (يدوياً):
1. قم بإنشاء حساب على [Railway.app](https://railway.app/).
2. قم بإنشاء مستودع (Repository) جديد على GitHub الخاص بك وارفعه عليه هذه الملفات.
3. في Railway، اختر **"New Project"** ثم **"Deploy from GitHub repo"**.
4. اختر المستودع الذي رفعته.
5. سيقوم Railway بتعرف على ملف `Procfile` وتشغيل البوت تلقائياً.

## 👨‍💻 أوامر المطور:
- لإضافة نقاط: `/add_points [user_id] [amount]`
- زر الإحصائيات يظهر فقط للمعرف: `7666967863`
